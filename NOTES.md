### 23. throughline-studio/ — Throughline Studio (**the shippable capstone**)
**A→Z research workspace** unifying the suite — a meta-layer over existing surfaces (honours subtract-not-add). React 19 + Vite + TS, hash router + hand-crafted design system (no Tailwind). `npm run dev` (:5180) · `build` (`tsc -b && vite build`).
- **Spine = `Project` through 7 stages** (`src/lib/stages.ts` `STAGES` registry, tool assignments there): **Discover → Frame → Measure → Collect → Analyze → Write → Publish**; each stage deep-links into the real deployed tools via `deepLink()` (mirrors hub `buildDeepLink` keys); inbound `?topic=` on a bare URL auto-creates a project (hub Quick Start leg). The 7 stage bodies are lazy chunks; route changes scroll-to-top; `.stage-grid` enters 70ms behind `.stage-head`.
- Core: `types.ts` (`Project`/`StageState`/`StageId`) · `store.ts` (`tls_projects`/`tls_active`+`progress()`) · `router.ts` (`#/`, `#/p/<id>[/<stage>]`, `#/examples[/<slug>]`) · `api.ts` → local `/api/generate`, **falls back to RF's shared Groq `/api/generate`** (normalizes `{result}`→data) on 404/503/network — suite AI works despite the gateway blocker. Views: Hub (state-aware — projects exist → slim `hero-compact`, no pitch) + ProjectView (Spine + StageShell + **Thread tray**: persistent strip of every stage's `carrySummary`, chips jump-navigate, hidden until artifacts exist).
- **All 7 stages embedded** (`src/stages/StageShell.tsx`): **Discover** = `lib/corpus.ts` streams Research Book JSON (BOM-stripped); reading list = slim `SavedPaper`s, never the corpus; opt-in **↑ Recent tier** (~42 MB, `localStorage.tls_recent`, sharded abstracts, badged "↑ recent · machine-tagged", true counts) + ⚡ live OpenAlex. **Frame** = question/gap/hypotheses/design + theory lens (`lib/theories.ts`) + **✦ Draft with AI** (empty fields ONLY, reading-list-grounded, no-fab + verify notice) + Research Compass gap link. **Measure** = composer over ScaleScope's 332 scales, **verbatim items inline** for the 178 reproducible (lazy `scale-items.json`). **Collect** = power-based N (`lib/power.ts`, displayed-default aware; warns when mediators/moderators exceed the d/r planner) + blueprint + **prereg assembled & LOCKED here, before fielding** (`lib/prereg.ts`, timestamped `preregLockedAt`; legacy Publish-stage plans migrate; Frame shows a locked banner) + **✦ Draft plans with AI** (3 plans, unlocked + empty-only) + ethics/IRB checklist (incl.
